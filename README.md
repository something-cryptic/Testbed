# YouTube Analyzer

AI-powered YouTube channel analytics. Connect your channel, fetch video metrics, and get actionable recommendations generated by Claude.

## Architecture

```
youtube-analyzer/
├── apps/
│   ├── api/          TypeScript Express API (port 8000)
│   ├── web/          React + Vite frontend (port 5173)
│   └── processor/    Python FastAPI microservice (port 8001)
└── packages/
    └── types/        Shared TypeScript interfaces
```

## Setup

### 1. Environment variables

Copy `.env.example` to `.env` and fill in your credentials:

```env
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
ANTHROPIC_API_KEY=...
REDIRECT_URI=http://localhost:8000/auth/callback
DATABASE_PATH=./data.db
FRONTEND_URL=http://localhost:5173
```

**Google Cloud Console setup:**
1. Create a project at [console.cloud.google.com](https://console.cloud.google.com)
2. Enable **YouTube Data API v3** and **YouTube Analytics API**
3. Create OAuth 2.0 credentials (Web application)
4. Add `http://localhost:8000/auth/callback` as an authorized redirect URI

### 2. Install dependencies

```bash
npm install
```

### 3. Python processor (optional but recommended)

```bash
cd apps/processor
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 4. Run everything

```bash
npm run dev
```

This starts all three services concurrently:
- API → http://localhost:8000
- Web → http://localhost:5173
- Processor → http://localhost:8001

If the Python processor is not running, the API falls back gracefully and Claude analyzes the raw video data directly.

## Usage

1. Open http://localhost:5173
2. Click **Sign in with Google** and authorize the app
3. Your videos load automatically on the dashboard
4. Click **Run Analysis** to generate AI recommendations
5. View detailed recommendations on the Recommendations page

| Layer | Tech |
|---|---|
| API | TypeScript, Express, better-sqlite3, googleapis, @anthropic-ai/sdk |
| Frontend | React 18, Vite, Tailwind CSS, Recharts, React Router |
| Processor | Python, FastAPI, uvicorn |
| AI | Claude (claude-opus-4-6) |
| Auth | Google OAuth 2.0 |

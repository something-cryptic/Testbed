# Content Analyzer

AI-powered multi-platform content analytics. Connect YouTube, Instagram, and Twitch, fetch performance metrics, and get actionable recommendations generated by Claude.

## Features

- **Multi-platform support** — YouTube, Instagram, and Twitch in one dashboard
- **AI analysis** — per-platform and holistic cross-platform recommendations powered by Claude
- **Analytics charts** — 30 interactive Recharts visualizations per platform (views, engagement, retention, upload frequency, and more)
- **10 color themes** — lavender, midnight, forest, rose, ocean, sunset, aurora, neon, ember, arctic
- **Profile & avatar** — custom avatar upload, username, per-user preferences
- **Mobile responsive** — bottom navigation bar on mobile, stacked cards, touch-friendly targets
- **Token management** — auto-refresh for all platforms, manual disconnect in Settings

## Setup

### 1. Environment variables

Create a `.env` file in the project root:

```env
# Google / YouTube OAuth
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
REDIRECT_URI=http://localhost:8000/auth/callback

# Meta / Instagram OAuth
META_APP_ID=
META_APP_SECRET=
INSTAGRAM_REDIRECT_URI=http://localhost:8000/auth/instagram/callback

# Twitch OAuth
TWITCH_CLIENT_ID=
TWITCH_CLIENT_SECRET=
TWITCH_REDIRECT_URI=http://localhost:8000/auth/twitch/callback

# Anthropic
ANTHROPIC_API_KEY=

# App
DATABASE_PATH=./analyzer.db
FRONTEND_URL=http://localhost:5173
```

### 2. Install dependencies

```bash
npm install
```

### 3. Python processor (optional)

The processor is used for pre-processing video data before Claude analysis. If it is not running the API falls back gracefully and sends raw data to Claude directly.

```bash
cd apps/processor
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 4. Run everything

```bash
npm run dev
```

Starts all three services concurrently:

| Service | URL |
|---|---|
| API | http://localhost:8000 |
| Web | http://localhost:5173 |
| Processor | http://localhost:8001 |

## Usage

1. Open http://localhost:5173
2. Click **Sign in with Google** to authenticate
3. Your YouTube data loads automatically on the Dashboard
4. Click **Analyze YouTube** (or any connected platform) to generate AI recommendations
5. View charts on the **Analytics** page — use the dropdown on each card to switch between 30+ chart types
6. View recommendations on the **Recommendations** page — filter by platform tab
7. Connect additional platforms from the **Platforms** page
8. Customize your theme, avatar, and preferences on the **Profile** page

## Platform Setup

### YouTube

1. Create a project at [console.cloud.google.com](https://console.cloud.google.com)
2. Enable **YouTube Data API v3** and **YouTube Analytics API**
3. Create OAuth 2.0 credentials (Web application)
4. Add `http://localhost:8000/auth/callback` as an authorized redirect URI
5. Copy **Client ID** and **Client Secret** to `.env`

### Instagram

Instagram requires the Meta Graph API with a Professional account (Creator or Business).

1. Go to [developers.facebook.com](https://developers.facebook.com) → **My Apps** → **Create App** (Consumer type)
2. Add product: **Instagram Graph API**
3. Under **Instagram → Basic Display**, set:
   - Valid OAuth Redirect URI: `http://localhost:8000/auth/instagram/callback`
4. Add yourself as a test user under **Roles → Test Users**
5. Switch your Instagram to a Professional account: **Settings → Account → Switch to Professional Account**
6. Copy **App ID** and **App Secret** to `.env`

### Twitch

1. Go to [dev.twitch.tv/console](https://dev.twitch.tv/console) → **Register Your Application**
2. Set OAuth Redirect URL to `http://localhost:8000/auth/twitch/callback`
3. Category: **Website Integration**
4. Copy **Client ID** and generate a **Client Secret**, add both to `.env`

## Token lifecycle

| Platform | Token type | Refresh strategy | Expiry |
|---|---|---|---|
| YouTube | OAuth 2.0 short-lived + refresh token | Automatic via googleapis | ~1 hour access / refresh persists |
| Instagram | Long-lived token (no refresh token) | Refreshed if within 7 days of expiry | 60 days |
| Twitch | OAuth 2.0 short-lived + refresh token | Automatic on expiry | ~4 hours access / refresh persists |

All tokens are stored encrypted in SQLite and refreshed transparently before API calls. Manual disconnect is available in **Settings** for any platform.

## Stack

| Layer | Tech |
|---|---|
| API | TypeScript, Express, better-sqlite3, googleapis, node-fetch |
| Frontend | React 18, Vite, Tailwind CSS, Recharts, React Router v6, Lucide |
| Processor | Python, FastAPI, uvicorn |
| AI | Claude (`claude-opus-4-6`) via `@anthropic-ai/sdk` |
| Auth | Google OAuth 2.0, Meta OAuth 2.0, Twitch OAuth 2.0 |
| Database | SQLite (better-sqlite3) with WAL mode |
